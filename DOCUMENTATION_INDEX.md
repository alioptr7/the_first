## üîó Key Links

### Command Reference
```bash
# Validate setup
./validate_setup.sh

# Start all services
docker-compose -f docker-compose.dev.yml up -d --profile all

# View logs
docker-compose logs -f api worker beat

# Run tests
pytest tests/

# Access services
# Response Network: http://localhost:8000
# Request Network: http://localhost:8001
# PostgreSQL Response: localhost:5432
# PostgreSQL Request: localhost:5433
# Redis: localhost:6380
# Elasticsearch: http://localhost:9200
```

### Service Ports
- **Response Network API:** 8000
- **Request Network API:** 8001
- **Response PostgreSQL:** 5432
- **Request PostgreSQL:** 5433
- **Redis:** 6380
- **Elasticsearch:** 9200
